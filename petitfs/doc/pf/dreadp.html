disk_readp

The disk_readp function reads a partial sector from the disk drive.

DRESULT disk_readp (
  void* Destination,   /* Pointer to the destination object */
  DWORD SectorNumber,  /* Sector number */
  WORD Offset,         /* Byte offset in the sector to start to read */
  WORD Count           /* Number of bytes to read and destination flag */
);

Parameters

Destination
Pointer to the destination object to put the read data. The destination object is either memory (void* buffer) or streaming function ((BYTE)(*func)(BYTE)). The type of the destination is specified by bit 15 in the Count.
SectorNumber
Specifies the sector number in logical block address (LBA).
Offset
Specifies the byte offset in the sector to start to read. The value can be 1 to 511.
Count
Bit14-0 specifies number of bytes to read. The value can be 1 to 512.
Bit15 specifies the type of the destination. 0: Memory, 1: Streaming function.

Return Value

RES_OK (0)
The function succeeded.
RES_ERROR
Any hard error occured during the disk read operation and could not recover it.
RES_STRERR
Any hard error occured in the streaming function.
RES_PARERR
Invalid parameter.
RES_NOTRDY
The disk drive has not been initialized.
