pf_read

The pf_read function reads data from a file.

FRESULT pf_read (
  void* Destination,  /* Pointer to the destination object */
  WORD ByteToRead,    /* Number of bytes to read */
  WORD* ByteRead      /* Pointer to the variable to return number of bytes read */
);

Parameters

Destination
Pointer to the destination object to put the read data.
ByteToRead
Number of bytes to read.
ByteRead
Pointer to the WORD variable to return number of bytes read.

Return Values

FR_OK (0)
The function succeeded.
FR_DISK_ERR
The function failed due to an error in the disk function, a wrong FAT structure or an internal error.
FR_NOT_READY
The disk drive cannot work due to no medium in the drive or any other reason.
FR_INVALID_OBJECT
The file has not been opened.

Description

The read pointer in the file system object increases in number of bytes read. After the function succeeded, *ByteRead should be checked to detect end of file. In case of *ByteRead < ByteToRead, it means the read pointer reached end of file during read operation.

The destination type is specified by FA_STREAM bit in the flag member in the file system object. When it is cleared, Destination points the read buffer (void* buffer). When it is set, Destination typically points the streaming function ((BYTE)(*func)(BYTE)), but it will not be used if streaming code is built in the dreadp function.
